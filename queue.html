QUEUE OPERATIONS

QUEUE SIZE = 5

Front: -
Rear: -

Enqueue Algorithm

Time Complexity: O(1)
Algorithm Enqueue(queue, rear, maxSize, newElement):
    if rear >= maxSize:
        print "Queue Overflow"
        return
    queue[rear] = newElement
    rear = rear + 1

Dequeue Algorithm

Time Complexity: O(1)
Algorithm Dequeue(queue, front, rear):
    if front == rear:
        print "Queue Underflow"
        return None
    element = queue[front]
    front = front + 1
    return element
