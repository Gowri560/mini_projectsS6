Push Algorithm

Time Complexity: O(1)
Algorithm Push(stack, top, maxSize, newElement):
    if top >= maxSize - 1:
        print "Stack Overflow"
        return
    top = top + 1
    stack[top] = newElement

Pop Algorithm

Time Complexity: O(1)
Algorithm Pop(stack, top):
    if top < 0:
        print "Stack Underflow"
        return None
    element = stack[top]
    top = top - 1
    return element
